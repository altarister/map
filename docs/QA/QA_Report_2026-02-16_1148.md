## 1. 주요 발견 사항 (Key Findings)

### ✅ 기능 구현 완성도 (Functionality)
- **GameOptionSelectScreen**: 레벨 탭, 난이도 토글, 챕터 그리드 UI가 기획 의도대로 구현됨.
- **단일 챕터 선택**: 챕터 클릭 시 시각적 피드백(Highlight/Dimmed)과 데이터 필터링이 정상 동작함.
- **난이도 시스템**: `NORMAL` (라벨 표시) / `HARD` (라벨 숨김) 모드가 정상 작동하며 `localStorage`에 상태가 유지됨.
- **숙련도 시스템**: 게임 결과에 따라 숙련도(%)가 계산되고 저장되며, 100% 달성 시 뱃지가 표시됨.

### 🐞 버그 및 수정 (Bugs & Fixes)

#### [FIXED] Critical: ActionBar 문제 텍스트 미표시
- **증상**: 게임 시작 시 상단 ActionBar에 문제 텍스트가 `Q. 다음 지역을 찾으세요: -` 형태로 표시됨.
- **원인**: `ActionBar` 컴포넌트가 `LOCATE_SINGLE` 타입만 처리하도록 하드코딩 되어 있었으나, 테스트 환경(Level 2 지속 상태)에서 `LOCATE_PAIR` 타입 문제가 출제됨.
- **조치**: `ActionBar.tsx`에 `getQuestionText` 헬퍼 함수를 추가하여 모든 문제 타입(`LOCATE_SINGLE`, `LOCATE_PAIR`)의 텍스트를 렌더링하도록 수정함.
- **검증**: 수정 후 브라우저 테스트에서 `Q. 다음 지역을 찾으세요: 화전동 ➡️ 흥도동` 형태로 정상 표시됨을 확인.
