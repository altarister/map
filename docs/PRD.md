# 프로젝트: 화물 배차 판단력 향상 시뮬레이터 (Freight Mapper MVP)

## 1. 프로젝트 개요

- **배경**: 화물 어플(24시 콜, 원콜 등)에서 콜이 떴을 때, 3초 이내에 지역 위치, 공차 거리, 적정 단가 여부를 판단해야 함.
- **목적**: 프론트엔드 기술을 활용하여 지리적 위치 암기와 수익성 판단 속도를 높이기 위한 트레이닝 툴 개발.
- **타겟**: 본인(승욱) 및 지리에 익숙하지 않은 초보 화물 차주.

## 2. 사용자 요구사항 (User Stories)

1. **지역 식별**: 지도의 행정구역 경계선을 보고 해당 지역이 어디인지 즉각적으로 알 수 있어야 함.
2. **위치 암기**: 랜덤하게 제시되는 지역명을 보고 지도에서 해당 위치를 정확히 클릭할 수 있어야 함.
3. **인접성 판단**: 현재 위치(또는 1차 목적지)와 새로운 상차지 간의 거리를 시각적으로 파악하여 합짐 가능 여부를 결정해야 함.
4. **수익성 계산**: 이동 거리 대비 제시된 단가가 '똥콜'인지 '꿀짐'인지 빠르게 계산할 수 있어야 함.

## 3. 핵심 기능 (MVP 단계)

### Phase 1: 위치 식별 및 하이라이트

- GeoJSON 데이터를 활용한 수도권(서울/경기/인천) 지도 렌더링.
- 지역(시/군/구) 클릭 시 해당 구역의 테두리(Stroke) 강조 및 이름 표시.

### Phase 2: 랜덤 지역 맞추기 퀴즈

- 시스템이 랜덤한 지역명(예: "안산시 단원구")을 제시.
- 사용자가 지도에서 올바른 위치를 클릭하면 정답 처리 및 다음 문제 진행.
- 오답 시 해당 지역의 정확한 명칭을 피드백으로 노출.

### Phase 3: 배차 시뮬레이션 (확장 예정)

- 가상 콜 공고 띄우기 (상차지, 하차지, 단가).
- 지도 위에 두 지점을 직선으로 연결하여 거리 시각화.
- 적정 단가와 비교하여 '잡을지 말지' 결정하는 인터페이스.

## 4. 학습 우선순위 로드맵

1. **[기초]** 지역 이름과 위치 매칭 (어디에 있는가?)
2. **[중급]** 지역 간 이동 경로 및 인접 구역 파악 (어떻게 가는가?)
3. **[심화]** 거리 대비 적정 단가 산출 및 수익성 판단 (얼마를 받는가?)
